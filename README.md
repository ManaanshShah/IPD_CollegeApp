**🌐 [View Live Deployment](https://ipd-college-app.vercel.app)**

---

## ✨ Key Features

* **Role-Based Access Control:** Distinct, secure experiences for Students and Faculty.
* **Unified Announcement Hub:** Faculty can instantly broadcast hackathons, internships, and general notices.
* **Dynamic Student Timetables:** Real-time, day-by-day class schedules mapped to specific student batches.
* **Live Link Integration:** Seamless external routing for internship applications and event registrations.

## 🛠️ Tech Stack

**Frontend Architecture:**
* React.js
* React Router DOM (Dynamic SPA Routing)
* Lucide-React (Iconography)
* Deployed via **Vercel**

**Backend & Database:**
* **Python / FastAPI:** High-performance API routing and validation.
* **Pydantic:** Strict schema validation for data integrity.
* **SQLAlchemy:** Object-Relational Mapping (ORM).
* **Supabase / PostgreSQL:** Relational database for persistent storage.
